# Clasificador de Mensajes - UIN

Este proyecto es una aplicación web para clasificar mensajes, construida con HTML, CSS, JavaScript y Vite.

## Requisitos Previos

Para ejecutar este proyecto en Windows (o cualquier otro sistema operativo), necesitas tener instalado **Node.js**.

1.  Descarga e instala Node.js desde [nodejs.org](https://nodejs.org/). Se recomienda la versión **LTS**.
2.  La instalación incluirá automáticamente `npm` (Node Package Manager).

## Cómo ejecutar en Windows

Sigue estos pasos una vez que tengas los archivos del proyecto en tu computadora:

1.  **Abre la Terminal o PowerShell:**
    *   Presiona `Win + R`, escribe `cmd` o `powershell` y presiona Enter.
    *   O navega a la carpeta del proyecto en el Explorador de Archivos, haz clic derecho en un espacio vacío y selecciona "Abrir en Terminal" (Open in Terminal).

2.  **Navega a la carpeta del proyecto:**
    *   Usa el comando `cd` seguido de la ruta de la carpeta. Por ejemplo:
        ```bash
        cd "C:\Ruta\A\Tu\Proyecto\Clasificador de mensajes - UIN"
        ```

3.  **Instala las dependencias:**
    *   Ejecuta el siguiente comando para descargar los módulos necesarios (esto solo se hace la primera vez):
        ```bash
        npm install
        ```

4.  **Inicia el servidor de desarrollo:**
    *   Ejecuta el siguiente comando para arrancar la aplicación:
        ```bash
        npm run dev
        ```

5.  **Abre la aplicación:**
    *   Una vez que la terminal muestre que el servidor está listo, verás una dirección como `http://localhost:5173/`.
    *   Copia esa dirección y pégala en tu navegador (Chrome, Edge, etc.).

## Notas adicionales

*   Si deseas realizar cambios en el código, la aplicación se actualizará automáticamente en el navegador mientras el comando `npm run dev` esté activo.
*   Para detener el servidor, presiona `Ctrl + C` en la terminal.
